# Contributing to TestPilot AI

Thank you for your interest in contributing to TestPilot AI!

## How to Contribute

### 1. Fork the Repository

- Click the "Fork" button in the top right corner of the repository
- This creates a copy of the repository under your account

### 2. Clone Your Fork

- Open your terminal and run:
  ```bash
  git clone https://github.com/YOUR_USERNAME/TestPilot-AI.git
  ```

### 3. Create a Feature Branch

- Create a new branch for your feature:
  ```bash
  git checkout -b feature/my-feature
  ```

### 4. Make Your Changes

Please follow the coding standards:
- Java 17
- Spring Boot best practices
- Clean Code principles
- SOLID principles

### 5. Test Your Changes

- Run the tests:
  ```bash
  mvn clean test
  ```
- Ensure all tests pass

### 6. Commit Your Changes

- Use meaningful commit messages:
  ```bash
  git commit -m "Added feature: description"
  ```

### 7. Push to Your Fork

- Push your branch:
  ```bash
  git push origin feature/my-feature
  ```

### 8. Open a Pull Request

- Describe your changes:
  - What changed
  - Why you made the changes
  - Screenshots (if UI changed)

## Coding Standards

- Use meaningful variable names
- Follow Java naming conventions
- Add JavaDoc for public methods
- Keep methods small and focused
- Avoid duplicated code

## Reporting Issues

Please include:
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots (if applicable)
- Logs or error messages
- Java version
- Environment details
